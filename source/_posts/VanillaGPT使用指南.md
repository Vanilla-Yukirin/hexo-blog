http://203.91.78.60:6017/

**记住你的密钥**，这个密钥很重要，有了密钥就可以直接访问GPT。由于我是使用API进行访问的，按照每次提问回答量（prompt）进行计算价格扣费，所以**不要告诉别人**。

## 登录

使用新设备，打开网站，随便发送一句话，会提示“请前往**登录**页输入正确的访问密码”，点击“登录”。

![](https://vanilla-picture-with-picgo.oss-cn-shenzhen.aliyuncs.com/img-from-picgo/20240823-011330.png)

在第二个框中输入你的密钥，点击确认。

![](https://vanilla-picture-with-picgo.oss-cn-shenzhen.aliyuncs.com/img-from-picgo/20240823-011533.png)

现在就实现了成功登录了。

点击下方的小机器人头按钮以切换模型。

模型很多，但是要用的舒服，请按照我推荐的模型来使用。

## 模型的推荐

| 模型名称                   | 简介                                   | 推荐用途                               | 价格比例 |
| -------------------------- | -------------------------------------- | -------------------------------------- | -------- |
| gpt-**3.5-turbo-1106**     | 比较老、小的模型                       | 不推荐，试试**4o-mini**吧              | 80       |
| gpt-**4-turbo**-2024-04-09 | **地表最强**                           | **正式用途时**                         | 2130     |
| gpt-**4o**                 | 没4-turbo那么强，但是**飞快**啊        | **中型问题**，或需要**迅速回答**的时候 | 1065     |
| gpt-**4o-mini**            | 4o的迷你版，超快超便宜                 | **小型问题**                           | 42.6     |
| gpt-**4o-2024-08-06**      | 比4o好（？），比4o便宜，**结构化输出** | 可以试试                               | 710      |
|                            |                                        |                                        |          |
| gpt-4                      | 最早的gpt-4模型                        | 贵！去用**4-turbo**                    | 4260     |
| gpt-4-1106-preview         | 2023/11/6发布的4，当然没有4-turbo好    | 不再推荐                               | 2130     |

未在上表中列出来的模型（特别是其他的那些带了个“-16K”、“-128K”或者其他日期的）尽量不要使用。上述模型均为目前最佳的解决方案。

总结：请在模型列表中**仔细地**找到：`gpt-4-turbo-2024-04-09`、`gpt-4o`、`gpt-4o-mini`这三个模型，**根据具体需求**进行使用。
